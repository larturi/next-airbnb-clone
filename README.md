# Airbnb Clone with Next.js

### Next 13, Prisma, MongoDB, Tailwind, NextAuth

## Prerequisites

Node version 14.x

## Install packages

```bash
npm i
```

## Setup .env file

```bash
DATABASE_URL=
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
GITHUB_ID=
GITHUB_SECRET=
NEXTAUTH_SECRET=
```

## Setup Prisma

```bash
npx prisma db push
```

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
